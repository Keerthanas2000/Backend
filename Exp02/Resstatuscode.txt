**Commonly Used HTTP Status Codes**

### 1xx: Informational Responses
- **100 Continue** â†’ Server has received the request headers and is waiting for the body.
- **101 Switching Protocols** â†’ Server agrees to switch to a different protocol (e.g., WebSockets).

### 2xx: Success
- **200 OK** â†’ Request was successful.
- **201 Created** â†’ A new resource was successfully created.
- **202 Accepted** â†’ Request has been accepted but not yet processed.
- **204 No Content** â†’ Request was successful, but thereâ€™s no content to return.

### 3xx: Redirection
- **301 Moved Permanently** â†’ The requested resource has been permanently moved to a new URL.
- **302 Found** â†’ Resource has been temporarily moved to another URL.
- **304 Not Modified** â†’ Resource has not been modified since the last request (used in caching).

### 4xx: Client Errors
- **400 Bad Request** â†’ The request is malformed or invalid.
- **401 Unauthorized** â†’ Authentication is required.
- **403 Forbidden** â†’ Request is valid but access is denied.
- **404 Not Found** â†’ The requested resource doesnâ€™t exist.
- **405 Method Not Allowed** â†’ HTTP method is not supported for the endpoint.
- **409 Conflict** â†’ Conflict with the current state of the resource.
- **429 Too Many Requests** â†’ Rate limit exceeded.

### 5xx: Server Errors
- **500 Internal Server Error** â†’ Generic server error.
- **502 Bad Gateway** â†’ Server received an invalid response from the upstream server.
- **503 Service Unavailable** â†’ Server is temporarily unavailable (maintenance or overload).
- **504 Gateway Timeout** â†’ Upstream server did not respond in time.
